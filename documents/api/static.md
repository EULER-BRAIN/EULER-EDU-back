# /static
static data나 이미지 제공을 지원하는 api

## /gallery/:id (GET)
### URL Parameters
 - id : 갤러리 사진 고유아이디
### Response
 - 갤러리 이미지 파일(mimetype: image)

## /campus/:id (GET)
### URL Parameters
 - id : 캠퍼스 고유아이디
### Response
 - 캠퍼스 이미지 파일(mimetype: image)

## /book/:id (GET)
### URL Parameters
 - id : Book 고유아이디
### Response
 - Book 이미지 파일(mimetype: image)
